# RECIPE-API
This project is a Recipe Management Web Application built using Node.js, Express, SQLite, and EJS. It allows users to view a list of recipes and interact with each recipe by adding comments and ratings.

#HOW TO RUN THE PROJECT
Clone the repository
in your terminal run "npm install"
then run "npm run dev"
in your browser, open https://localhost:3000.
